# Data Sources

All files in this directory are synthetic. They are generated by `scripts/score_operating_data.py` with seed `20260519`.

The data is modeled on mobile programmatic advertising workflows where analysts review campaign delivery, bidding signals, creative performance, mobile attribution quality, and client recommendations. The synthetic generator uses budget ranges, campaign objectives, channel factors, contextual fit, signal coverage, win-rate gaps, creative fatigue, and measurement reliability to produce plausible but fictional records.

Generated files:

- `campaigns.csv`
- `daily_campaign_metrics.csv`
- `contextual_bid_signals.csv`
- `creative_variants.csv`
- `measurement_qa_checks.csv`
- `client_recommendations.csv`
